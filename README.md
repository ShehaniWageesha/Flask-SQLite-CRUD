# API REST - Flask, Python, SQLite3, JSON

The capabilities of this application include
- Create New Users.
- User Login.
- User Verification.
- User Profile.
- Add Courses.

## Clone the repository
```sh
git clone https://github.com/ShehaniWageesha/Flask-SQLite-CRUD.git
```

## Run the server
```sh
python main.py
```
